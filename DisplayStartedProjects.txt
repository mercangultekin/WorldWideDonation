
Use Case Name: DisplayStartedProjects
Participating Actors : User as initiator
Communicates With WWDS
Entry Condition : User should click on "Display Profile"
Flow of Events : 
1) User clicks "Display Started Projects".
  2) System displays the list of the project that user has started.  
Exit Condition : User successfully views the list of the projects that s/he started.
Quality Requirements : This use case includes "DisplayProfile". When "DisplayProfile" initated user can view his/her 
profile page including "DisplayOngoingDonations","DisplayMyFinishedProjects","DisplayDonatedProjects","DsiplayStartedProjects"
