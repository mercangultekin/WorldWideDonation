UseCase : ResetPassWord
Participating Actors : User as Participant
Entry Condition: User should be logged in
                 User clicks on "Display Profile"
 
Flow of Events : 
1) User enters his account page.
        2)WDS system directs User to profile page. Display tabs such as "account settings" button,
	   "Ongoing Donations" button, "Display My Finished Projects" button,"Display Donated Projects" button,
	   "Display Started Projects" button.
3) User clicks on "account settings" button.
        4)WWDS displays a form. In form WWDS ask about new password.
5) User enters the new password
       
Exit Condition: User must enters the new password as she/he wishes successfully.
