Use Case Name:Display Cargo List of Project

Participating Actor(s):User-Initiates

Entry Condition(s):
1)User must first use Display Started Projects Use Case

Exit Condition:Display the list of cargos sent for this project.

Flow of Events:

1)User clicks on "Display Cargo List Button" in the list of his/her started projects
	2)System displays the list of cargos that sent for this project in two lists as sent and delivered.
